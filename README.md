## How to Use This Repo

- **Browse on GitHub** — every folder has a `README.md` that GitHub renders automatically; click through the tree
- **Quick review** — open any topic file for bullet-point summaries, comparison tables, and interview-ready answers at the bottom
- **Deep-dive** — open `kafka-knowledge-graph.html` locally or via GitHub Pages for a single-page interactive view with progress tracking
